[Jetson Orin Nano PyTorch CUDA 환경 설정 과정]

1. 시스템 환경 확인
   - Jetson Linux: R36.4.3
   - CUDA: 12.6
   - Python: 3.10
   - Architecture: aarch64

2. 기존 문제 확인
   기존에 설치된 PyTorch:
   - torch 2.12.1+cu130
   - PyTorch CUDA build: 13.0

   하지만 Jetson 시스템의 CUDA 환경은 12.6이므로,
   CUDA 13.0용 PyTorch와 Jetson의 CUDA 12.6 드라이버가 호환되지 않아
   torch.cuda.is_available()이 False로 반환됨.

3. 기존 PyTorch 제거

   python -m pip uninstall -y torch torchvision torchaudio

4. NumPy 호환 버전 설치

   python -m pip install --no-cache-dir "numpy<2"

   설치 결과:
   - numpy 1.26.4

5. Jetson CUDA 12.6 환경에 맞는 PyTorch 설치

   python -m pip install --no-cache-dir \
       torch==2.8.0 \
       --index-url https://pypi.jetson-ai-lab.io/jp6/cu126

   기존에 시도한 torch 2.7.0은 해당 저장소에서 제공되지 않아
   현재 사용 가능한 CUDA 12.6용 torch 2.8.0을 설치함.

6. PyTorch CUDA 환경 확인

   import torch

   print("torch version:", torch.__version__)
   print("torch CUDA:", torch.version.cuda)
   print("CUDA available:", torch.cuda.is_available())
   print("device count:", torch.cuda.device_count())

   if torch.cuda.is_available():
       print("GPU:", torch.cuda.get_device_name(0))

7. 최종 확인 결과

   torch version: 2.8.0
   torch CUDA: 12.6
   CUDA available: True
   device count: 1
   GPU: Orin

8. 결론

   PyTorch 2.8.0과 CUDA 12.6의 버전이 일치하고,
   PyTorch에서 Jetson Orin GPU를 정상적으로 인식함.

   따라서 현재 환경에서는 PyTorch 기반 CUDA 연산이 가능한 상태임.
